# Diagram Rendering

The markdown docs in this repository use [Mermaid](https://mermaid.js.org/) syntax (` ```mermaid ` blocks) for diagrams.

## Viewing

Mermaid diagrams render natively on GitHub, most markdown editors, and documentation tools. No build step required.

## Supported Diagram Types

- Flowcharts
- State diagrams
- Sequence diagrams
- Class diagrams
- ER diagrams
- Gantt charts

## Source of truth

Diagram source lives in the markdown files alongside the documentation it describes.
