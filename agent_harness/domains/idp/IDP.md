# IDP Domain Memory

## Overview
Intelligent Document Processing — assists with plugin-based document extraction, classification, and validation.

## Core Functions
- Test standard identification from product description
- Test plan generation based on applicable standards
- Lab capability matching for required tests
- Cost and timeline estimation

## Applicable Standards
- ASTM, compliance, EN, ISO test standards
- Product-specific test requirements
- Regional compliance variations (US, EU, APAC)
